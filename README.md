friendChat
==================

Backend to simple chat to communicate with friends

## Setup

Customize your data in [config file](https://github.com/lukewalczak/friendChat/blob/backend/api/src/config/config.js)

## Installation

1. Install server
    `cd api && yarn`

## Usage

1. Run mongodb
`cd api && sudo mongod`

2. Run server
`cd api && npm run start`
