friendChat
==================

Simple chat to communicate with friends

## Setup

Customize your data in [config file](https://github.com/lukewalczak/friendChat/tree/master/api/src/config.js)

## Installation

1. Install server
    `cd api && yarn`

2. Install app
    `yarn`

## Usage

1. Run mongodb
`cd api && sudo mongod`

2. Run server
`cd api && npm run start`

3. Run app on ios or android
`npm run ios` or `npm run android`
